## Symbol Codex System

Every SPR now has a complete `symbol_codex` mapping symbols to meanings:

```json
{
  "Γ": {"meaning": "Cognitive", "context": "Protocol Symbol"},
  "Σ": {"meaning": "System", "context": "Protocol Symbol"},
  "Α": {"meaning": "Action", "context": "Protocol Symbol"},
  ...
}
```

**Decompression Process:**
1. Read `zepto_spr`: `"Γ|Σ|Α"`
2. Lookup in `symbol_codex`: `{"Γ": "Cognitive", "Σ": "System", "Α": "Action"}`
3. Reconstruct meaning: "Cognitive System Action"
4. **Time:** <1ms (pure dictionary lookup)
5. **Cost:** $0 (no LLM)
